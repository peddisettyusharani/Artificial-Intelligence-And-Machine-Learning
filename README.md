# 🌊 AI-Powered Flood Prediction System

A Machine Learning-based Flood Prediction System developed as part of the **SmartBridge – Rising Waters** project. The system predicts flood risk using historical weather data and multiple classification algorithms. The best-performing model is integrated into a Flask web application, providing an intuitive interface for flood risk assessment and supporting early disaster management.

---

## 📌 Project Overview

Floods are among the most devastating natural disasters, causing significant loss of life and property every year. This project leverages Artificial Intelligence and Machine Learning to analyze meteorological features such as annual rainfall, seasonal rainfall, and cloud visibility to predict the likelihood of flood events.

Multiple classification models—Decision Tree, Random Forest, K-Nearest Neighbours (KNN), and XGBoost—are trained and evaluated. The best-performing model is deployed through a Flask web application for real-time flood prediction.
